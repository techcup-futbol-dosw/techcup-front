# Modulo users

Responsabilidad:
- Gestion de perfil de usuario y configuraciones personales.
- Adaptar experiencia segun contexto (usuario, arbitro, organizador).

Pagina clave:
- `Profile.tsx`: datos personales, actividad y cambios de seguridad.

Criterios de mantenimiento:
- Mantener validaciones de contrasena y mensajes de error consistentes.
- Evitar duplicacion de reglas de negocio de contexto.
- Preservar feedback inmediato tras acciones de guardado.
